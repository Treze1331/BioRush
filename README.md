# Jogo de Biologia

Aplicação web estática com quiz multiplayer usando Supabase.

## Como hospedar

Este projeto pode ser hospedado em qualquer servidor estático como:
- Vercel
- Netlify
- GitHub Pages
- Apache/Nginx

### Arquivos de configuração
- `vercel.json` para Vercel
- `netlify.toml` para Netlify

### Execução local
```bash
python -m http.server 8000
```

Acesse http://127.0.0.1:8000/
